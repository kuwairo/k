# k

OCI image providing tools for working with Kubernetes.

## Tools

At the moment, the following tools are included in the image:
- `kubectl`
- `helm`
- `flux`
- `cilium`
- `k9s`

## Supported platforms

The image is built for the following platforms:
- `linux/amd64`
- `linux/arm64`
